# bootcamp-reactive-blog

## Create docker image "docker build"
`docker build -t bootcamp-blog:0.0.1 .`

`docker image ls`

## Crear contenedor "docker run"

`docker run -d --name bootcamp-blog-dev -p 9090:8080 bootcamp-blog:0.0.1`
